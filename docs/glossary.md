# 技术术语表

这里记录常见技术术语的简短解释，后续可以按字母或技术领域继续扩展。

| 术语 | 说明 |
|---|---|
| ADB | Android Debug Bridge，用于调试 Android 设备和模拟器的命令行工具。 |
| AGP | Android Gradle Plugin，Android 工程的 Gradle 插件。 |
| ANR | Application Not Responding，应用主线程长时间无响应。 |
| ART | Android Runtime，Android 5.0 之后默认运行时。 |
| Binder | Android 跨进程通信机制。 |
| GC | Garbage Collection，垃圾回收。 |
| JIT | Just-In-Time Compilation，即时编译。 |
| AOT | Ahead-Of-Time Compilation，预先编译。 |
| JMM | Java Memory Model，Java 内存模型。 |
| LLM | Large Language Model，大语言模型。 |
| MVVM | Model-View-ViewModel，常见应用架构模式。 |
| MVI | Model-View-Intent，强调单向数据流的架构模式。 |
| TCP | Transmission Control Protocol，传输控制协议。 |
| TLS | Transport Layer Security，传输层安全协议。 |
